Web proqramlaşdırma nədir?

Web proqramlaşdırma qısa desək, bir internet səhifəsində istifadəçinin gördüyü və görmədiyi, hər şey kimi deyə bilərik. Web proqramlaşdlrma iki yerə ayrılır:
Frontend və Backend

Front-End proqramlaşdırma nədir?

Front-End saytda istifadəçinin gördüyü vizual tərəfdir. Muasir web dünyasında, hər gün daha da əhəmiyyəti artır.

Front-End proqramlaşdırma dilləri hansılardır?

Front-End dekikdə HTML, CSS və JavaScript kimi, texnologiyalar nəzərdə tutulur. Proqramlaşdırma dünyasında HTML və CSS bir çoxları tərəfindən proqramlaşdırma dili kimi sayılmasa da, istifadəçi təcrübəsinin bu qədər əhəmiyyətli olduğu dövrdə, web proqramlaşdırma dünyasındakı yerlərini qorumaqdadırlar.

HTML (Hypertext Markup Language), bir saytın strukturunu müəyyənləşdirən, işarələmə dilidir. CSS (Cascading Style Sheets) ilə, saytın görünüşü dizayn edilir. JavaScript ilə isə, saytın istifadəçi interfeysindəki funksionallığı reallaşdırılır. Eyni zamanda Front-End tərəfində, CSS üçün Twitter tərəfindən yaradılmış, Bootstrap framework’ü və Javascript ailəsinə daxil olan Ajax,Angular JS kimi texnologiyalar da istifadə olunur

Back-End proqramlaşdırma nədir?

Back-End proqramlaşdırma daha asan bir formada desək, istifadəçinin görmədiyi daha mürəkkəb prosseslərin reallaşması üçün yaradılan, arxa plan strukturudur. Bir saytın server side’a aid, əlaqəli bütün prosesləri, eyni zamanda istifadəçinin daxil etdiyi məlumatın işlənməsini, Back-End tərərfi reallaşdırır. Back-end proqramlaşdırmada PHP, ASP.NET, JAVA, PHYTON, C# və.s. kimi proqramlaşdırma dilləri istifadə olunur
